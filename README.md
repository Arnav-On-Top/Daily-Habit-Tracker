A Daily Habit Tracker made by a beginner

index.html contains the structure
script.js contains how the application will work.
style.css contains the designing.

The application taken the name of the habit. when u press add, a calendar type windown is created with the name of the habit. clicking the tick option will mark as completed for the day.

This application wont reset if u close it or refresh it.

Application to be furthur improved with possible more features in the future.
